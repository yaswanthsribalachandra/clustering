# Clustering Algorithms (Unsupervised Learning)

This repository contains implementations and concepts of clustering algorithms in Machine Learning. It focuses on grouping data points based on similarity without using labeled data.

---

## Overview

Clustering is a type of Unsupervised Learning used to identify hidden patterns or groupings in data. This repository includes multiple clustering techniques along with practical implementations and visualizations.

---

##  Algorithms Covered

### K-Means Clustering
- Partition-based clustering algorithm  
- Groups data into K clusters  
- Uses centroids for cluster formation  

---

### Hierarchical Clustering
- Builds clusters in a tree-like structure  
- Agglomerative (bottom-up approach)  
- Dendrogram visualization  

---

### DBSCAN (Density-Based Clustering)
- Groups based on density of data points  
- Detects noise and outliers  
- No need to specify number of clusters  

---

## Concepts Covered

- Distance Metrics (Euclidean, Manhattan)  
- Elbow Method  
- Silhouette Score  
- Cluster Evaluation Techniques  
- Handling Noise & Outliers
